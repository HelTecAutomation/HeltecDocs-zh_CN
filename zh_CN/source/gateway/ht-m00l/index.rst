HT-M00L单通道LoRa网关
=====================

HT-M00L为惠利特与腾讯联名款LoRa网关。HT-M00L是一个小体积、低成本的单通道LoRa网关，使用Type-C接口。

-  由ESP32驱动SX1278芯片；
-  软件混频器（基带仿真程序），实现对125KHz SF7~SF12扩频因子的监听；
-  支持LoRaWAN协议1.0.2 Class A, Class C;
-  内置天线；



.. note::

   HT-M00L没有防水设计，使用时应注意避免与水接触。
